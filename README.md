# Customer_behaviour_analysis
Data Analytics project on customer behaviour analysis using Python, MySQL, and Power BI.

Data Analytics Project
Overview

This project demonstrates a complete data analytics workflow, starting from raw data and ending with business insights. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard creation, and presentation of findings.

Dataset

The project uses a structured dataset containing business-related information. The data is analysed to identify trends, patterns, and key performance indicators (KPIs).

Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
Jupyter Notebook
SQL (PostgreSQL / MySQL / SQL Server)
Power BI
Gamma (Presentation)
Git & GitHub
Project Steps
1. Data Loading
Imported the dataset into Python.
Inspected data structure and data types.
Checked for missing values and duplicates.
2. Exploratory Data Analysis (EDA)
Generated descriptive statistics.
Explored feature distributions.
Identified trends, correlations, and outliers.
Created visualisations to understand the data.
3. Data Cleaning
Removed duplicate records.
Handled missing values.
Corrected inconsistent data.
Converted columns to appropriate data types.
4. SQL Analysis
Imported cleaned data into PostgreSQL/MySQL/SQL Server.
Wrote SQL queries to:
Filter and sort records
Aggregate data
Perform joins
Use GROUP BY and HAVING
Create business reports using SQL
5. Power BI Dashboard
Connected Power BI to the processed dataset.
Built an interactive dashboard with:
KPI cards
Charts and graphs
Filters and slicers
Trend analysis
6. Reporting & Presentation
Prepared a project report summarising the analysis.
Created a presentation using Gamma to communicate key findings and business insights.
Dashboard

Results

The project successfully:

Cleaned and prepared the dataset for analysis.

Built an interactive Power BI dashboard.
Documented findings in a report and presentation.
Demonstrated an end-to-end data analytics workflow.
How to Run
Clone the repository.
Install the required Python libraries.
Open the Jupyter Notebook.
Run the notebook from top to bottom.
Import the cleaned dataset into PostgreSQL/MySQL/SQL Server and execute the SQL scripts.
Open the Power BI file to explore the dashboard.
Review the report and presentation for project insights.
